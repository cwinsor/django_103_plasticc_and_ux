Building The Model

PLAsTiCC is a timeseries dataset. Here's the process to create a model:

  • Preprocessing establishes cleans up the data and establishes base attributes.
  • Feature Engineering is the task of creating features. Thousands of features (mostly hand-crafted) are common in PLAsTiCC submissions.
  • Feature Engineering eliminates the time domain, replaced by reduction features (think of "period" replacing a series of amplitudes).
  • A metric measures the model's performance. Metrics need to be established up front!

    Google's Tensorflow provides the framework.
