Why Make a Game?

Answer: To demonstrate how user's website experience can be enriched by careful application of Machine Learning.

In the game the user is given a difficult task: classify a SuperNova using astronomical timeseries datapoints.

To help, experts are "on hand" to offer advice. This advice comes from machine learning models which are the result of decades of astronomical study and petabytes of data analysis.

Provided in an understandable form and at the right time, the advice of experts (from the machine learning models) help the user win the game.
