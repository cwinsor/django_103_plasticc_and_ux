Why Make a Game?

The intent is to demonstrate how machine learning can be used to enrich a user's web-application experience.

In the game the user is tasked with classifying a star. Is it a:

  • Supernova?
  • Pulsar?
  • Binary-Pair?

    Experts, in the form of a machine learning model, are 'on hand' to help. They bring decades of experience and a rich knowledge based on deep data analysis. All this is made easily accessable to the user

    To help the user win!
