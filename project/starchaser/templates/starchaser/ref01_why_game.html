Why Make a Game?

The game demonstrates how Machine Learning can be integrated into a web application to enrich the user's experience.

In the game the user is given a difficult task: classify a SuperNova using astronomical timeseries datapoints.

Experts are "on hand" to offer advice. This advice is actually a set of machine learning models reflecting decades of astronomical study and petabytes of data analysis.

Integrated into the flow of the game, the models' recommendations are provided to the user in a natural and meaningful way. This helps the user make the classification, and therby win the game !
